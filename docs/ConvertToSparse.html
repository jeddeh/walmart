Owner
tqchen commented on Dec 30, 2014

Yes, though this is a bit extreme example. Another example could be red,green,red.
Y column is 1, 1, 0

Then the transformed one will be

  • red: 0.5
  • green: 1
  • red: 0.5

In another words, this is an empirical estimation of conditional expectations and other statistics. There are also smoothing tricks involved when you estimate the conditional expectations like normal Naive Bayes estimation.

Being said, I think one-hot encoding is sufficient for many cases and we can start from documenting one-hot encoding

Tianqi
